# Quick Start

```bash
macos/scripts/generate_icon.sh
macos/scripts/build_app.sh
open /Users/rsln-ua/Projects/monitor-backlight-service/dist/MonitorBacklight.app
```

Optional installer image:

```bash
macos/scripts/build_dmg.sh
```

First run:

1. Open `Control Center...`
2. Set `Govee IP` (required)
3. Set `Kasa 1 IP` and `Kasa 2 IP` (optional)
4. Click `Save`
5. Test with `Turn On -> All`
